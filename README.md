# OG02Homework
 *Репозиторий с файлами арифметическим 
 вычисленям и работой со строками*

В данном репозитории содержится проект 
со следующим функционалом 
(действия с двумя введенными числами):<br/>
* суммируются
* вычитаются
* умножаются
* делятся
* возведение числа 1 в степень числа 2

**Результаты выводятся на консоль.**

Также есть отдельный [файл работы со строками](str_conc.py) - <br/>
реализуется конкатенация введенных двух строк и вывод результата.

***Проект реализует успешные 
арифметические действия и конкатенацию строк*** <br/>
